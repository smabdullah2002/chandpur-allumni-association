# Services

This project includes a React frontend, a Node/Express API, MongoDB storage (via Mongoose), Cloudinary file storage, and SMTP email (Nodemailer).

## Backend API services
1. Auth: registration with document uploads, login, profile updates, password changes, phone privacy toggle.
2. Members: approved members listing with phone visibility rules.
3. Donations: submit and view donations; admin approval/rejection.
4. Notices: list notices for members; admin CRUD.
5. Events: public events listing; admin CRUD.
6. Gallery: public gallery listing; admin upload/delete.
7. Sliders: public hero slider listing; admin upload/delete.
8. About: public about content; admin edit.
9. Newsletter: email subscription; admin view/manage.
10. Stats: admin dashboard counts.

## Frontend app services
1. Public site: Home, About, Gallery, Events, Contact, Register, Login.
2. Member portal: Dashboard, Donations, Notices, Profile.
3. Admin portal: Users approvals, Donations review, Notices, Events, Fee Categories, Sliders, Gallery, About, Stats.
